# CelebrityWall

### 1) Setup the project

install pipenv
```bash
pip install pipenv
```

install python packages

```bash
pipenv install --dev
```

### 2) Set requirement variables
- Create `.env` same as `.env.example`

- Change `username` in `main.py` (Note: until we have list of Celebrities)

### 3) run the script

```bash
pipenv shell
python main.py
```
